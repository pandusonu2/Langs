MVC=Model, View, Controller

app/ 	Core app code-models,views and controllers and helpers
app/assests			CSS/JS and images
config/				Application configuration
db/					Database files
doc/ 				Documentation for the app
lib/				Library modules
lib/assests 		Library assests like CSS/JS and img files
log/				Application log files
pubic/				Databases accessible to the public
script/rails		script for generating code, opening console sessions, or starting a local server
test/				Application tests
tmp/				Temporary files
vendor/				Third party code such as plugins and gems
vendor/assests		Third party assests like CSS/JS and img files
README.rdoc			Readme files
Rakefile			Utility tasks available via rake command
Gemfile				Gem req for the app
Gemfile.lock		list of gem used to ensure that all copies of the app use same gem
config.ru			A config file for Rake middleware
